# Usage

### Clone Repo

```sh
git clone https://github.com/muhammadisa-n/be-tugas-akhir.git
```

### Or clone repo another branch

```sh
git clone -b namebranch https://github.com/muhammadisa-n/be-tugas-akhir.git
```

### Project Setup

```sh
npm install
```

### Configure database

```sh
npx prisma init
```

```sh
npx prisma migrate dev
```

### Run Server

```sh
npm run dev
```
